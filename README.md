# SQL Project Data Analysis
## Project Overview

This project analyzes the Olist E-commerce dataset to uncover insights into customer behavior, sales performance, product demand, and business growth opportunities.
Using SQL for data querying and Python (Pandas, Matplotlib) for analysis and visualization, the project delivers actionable insights to support data-driven decision-making.

## Objectives
- Analyze sales trends over time
- Identify top-performing products, cities, and states
- Understand customer behavior and retention
- Evaluate payment methods and delivery performance
- Generate strategic business recommendations

## Dataset Description
The project uses 7 datasets:
- customers – Customer information
- geolocation – Customer location data
- orders – Order details
- order_items_clean – Product-level order data
- payments – Payment information
- products – Product categories
- sellers – Seller details

## Tools & Technologies
- SQL (MySQL) – Data querying and transformation
- Python – Data analysis and visualization
- Pandas – Data manipulation
- Matplotlib – Data visualization
- Google Colab – Development environment

## Key Analysis & Insights
### Sales Trend
Sales peaked in October–November 2017
Strong growth observed through 2018
Indicates seasonal demand patterns

### Top Product Categories
Highest revenue categories:
- Bed, Table & Bath
- Health & Beauty
- Computer Accessories
These categories drive the majority of sales

### Payment Behavior
- Credit card dominates revenue (~70%)
- ~50% of orders are paid in installments
- UPI adoption is growing

### Delivery Performance
- Majority of deliveries are completed within 0–22 days
- Minimal delays beyond 40 days

### Seller Performance
- Revenue is concentrated among top-performing sellers
- Opportunity to improve the performance of smaller sellers

### Customer Retention
- Retention rate: ~2.3%
- Indicates strong reliance on one-time customers

### Geographic Insights
- Top States: Sao Paulo, Rio de Janeiro, Minas Gerais
- Top Cities: Sao Paulo, Rio de Janeiro, Belo Horizonte
- Revenue is highly concentrated in urban regions

### Customer Value Insights
- Average Order Value (AOV): 205.83 BRL
- High-value customers contribute significantly to revenue
- Some states show higher revenue per customer, indicating strong purchasing power

### Key Findings
- Revenue is concentrated in specific regions and categories
- Customers are price-sensitive
- Credit cards and installments drive most transactions
- Business has low retention but high acquisition
- A small segment of customers generates high lifetime value

### Recommendations
- Improve customer retention through loyalty programs
- Focus marketing on high-performing cities and states
- Optimize pricing strategies for high-demand products
- Encourage repeat purchases through personalized offers
- Support and incentivize underperforming sellers

### Project Deliverables
- SQL file (all queries)
- Python/Colab notebook (analysis & visualizations)
- Presentation (business insights)

### How to Run
- Clone the repository
- Open the SQL file in MySQL
- Run queries to generate insights
- Open the Colab notebook to view visualizations

### Conclusion
This project highlights how data can be used to uncover customer behavior, sales drivers, and growth opportunities. While the business shows strong sales performance, improving customer retention and expanding into underperforming regions will be key to long-term success.
